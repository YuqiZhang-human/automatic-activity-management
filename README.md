<p><h1 align="center">校级活动管理系统</h1></p>



## 项目介绍
基于springboot的校级活动管理系统：前端 vue2、elementui，后端 maven、springmvc、spring、mybatis；角色分为管理员、用户；集成活动信息、活动心得、公告信息、活动报名、收藏、评论等功能于一体的系统。

## 环境

- <b>IntelliJ IDEA 2021.3</b>

- <b>Mysql 5.7.26</b>

- <b>Node 14.14.0</b>

- <b>JDK 1.8</b>
